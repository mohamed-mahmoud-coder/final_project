



# Gaming Hub - Tournament Management System 

**Gaming Hub** is a comprehensive eSports tournament management platform developed using **PHP** and **MySQL**. This system allows administrators to organize tournaments while providing players with a seamless way to discover and join competitive events.

## Key Features

* **Admin Dashboard**: Empower admins to create, update, and delete tournaments, as well as manage registered participants.
* **User Authentication**: Secure registration and login systems for players to manage their profiles.
* **Tournament Management**: Real-time display of active tournaments, including prize pools, dates, and registration status.
* **Dynamic Database**: Automated data handling using MySQL to manage relationships between users and events.
* **Responsive UI**: A modern, sleek design built with **Bootstrap** to ensure compatibility across all devices.

## 🛠 Tech Stack

* **Backend**: PHP (Native)
* **Database**: MySQL / phpMyAdmin
* **Frontend**: HTML5, CSS3, JavaScript
* **Styling**: Bootstrap

## 📁 Project Structure

* `admin_home.php`: The main dashboard for administrators.
* `home.php`: The landing page for registered players.
* `connect.php`: Database connection configuration.
* `pr_db.sql`: The database schema file for easy setup.
* `registration.php` & `login.php`: Core files for handling user sessions.

## ⚙️ How to Run Locally

1. **Clone the repository**: Download the project files to your local machine.
2. **Move to Server Folder**: Place the files in your local server directory (e.g., `htdocs` for XAMPP).
3. **Setup Database**:
* Open **phpMyAdmin**.
* Create a new database (refer to `connect.php` for the expected name).
* Import the `pr_db.sql` file provided in the repository.


4. **Configure Connection**: Verify your database credentials in `connect.php`.
5. **Launch**: Open your browser and navigate to `localhost/final_project/index.php`.
